# Support

Use GitHub Discussions for usage questions and GitHub Issues for reproducible
bugs or scoped feature requests. Include Toolbelt version, Python version,
platform, command, exit code, redacted JSON output, and a minimal repository
shape. Run `toolbelt doctor --path PROJECT --json` when relevant.

Do not post secrets, private source, `.toolbelt/state.sqlite3`, or recovery
backups. Security issues belong in the private process described in
[SECURITY.md](SECURITY.md).
