# Support

Use [GitHub Discussions](https://github.com/H-Lupercal/agent-tooling/discussions) for
usage questions and design discussion. Use
[GitHub Issues](https://github.com/H-Lupercal/agent-tooling/issues) for reproducible
bugs and feature requests.

Include the affected project, package version, Python version, operating system,
command, sanitized output, and a minimal reproduction. Do not include credentials,
private source code, provider transcripts, or state databases. Report security issues
privately according to [SECURITY.md](SECURITY.md).
